# Cost System for 3D Printing
 
> Trabalho conclusivo do 3º semestre - Programação Orientada a Objetos
 
## Sobre o projeto
 
Sistema de cálculo de custos para impressões 3D, desenvolvido como trabalho conclusivo da disciplina de **Programação Orientada a Objetos** do curso de **Tecnologia em Sistemas para Internet (TSI)** no **IFSP — Campus Itapetininga**, sob orientação do professor **Gerson Nuno Carriel**.
